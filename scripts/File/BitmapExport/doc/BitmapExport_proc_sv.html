En dialogruta för att välja utdatafil visas. När filnamnet har angetts frågar en andra dialogruta efter bitmappstorlek och önskad bakgrundsfärg.
Observera att export till bitmappar med mycket stor storlek kan ta lång tid, beroende på din maskinvara. Den maximala storleken för bitmaps är bredd x höjd ≤ 2 147 483 647.
