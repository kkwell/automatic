Создайте новые пустые чертежи с помощью этой команды. Новые чертежи абсолютно пусты (нет сущностей, только один слой по умолчанию, нет блоков). Если вы часто пользуетесь QCAD, то вместо этого вы можете захотеть создать шаблоны чертежей с наиболее часто используемыми слоями и блоками и, возможно, границей чертежа. Создав такой шаблон, вы можете загрузить его вместо того, чтобы начинать новый чертеж с нуля.
Чтобы узнать, как использовать шаблоны чертежей, обратитесь к команде Новый из шаблона
