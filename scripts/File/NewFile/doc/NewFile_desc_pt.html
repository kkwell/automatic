Criar desenhos novos e vazios usando este comando. Novos desenhos estão completamente vazios (sem entidades, apenas uma camada por defeito, sem blocos). Como utilizador frequente do QCAD, poderá em vez disso querer criar modelos de desenho com as camadas e blocos que mais se utilizam e provavelmente uma borda de desenho. Uma vez que tenha tal modelo, pode carregar que em vez de começar um novo desenho a partir do zero.
Consulte também o comando New from Template para aprender a utilizar os modelos de desenho
