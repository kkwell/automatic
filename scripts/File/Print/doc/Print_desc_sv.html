Använd det här kommandot för att skriva ut en ritning. Se till att du alltid växlar till förhandsgranskningen av utskrift innan du skriver ut för att kontrollera om ritningens skala och position är inställd som önskat.
En inställningsdialog visas innan något skrivs ut. Använd den här dialogrutan för att välja skrivare och justera skrivarspecifika inställningar.
