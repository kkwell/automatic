O comando paste insere as entidades que foram previamente copiadas para a prancheta. A cola é especialmente útil para transferir entidades de um desenho para outro.
Todas as camadas que são utilizadas pelo conteúdo da prancheta são coladas na lista de camadas do desenho actual. As camadas existentes com o mesmo nome podem ser sobrescritas se desejado (opção "sobrescrever camadas" na barra de ferramentas de opções).
As referências dos blocos que estão na área de transferência serão coladas juntamente com as definições dos blocos a que se referem. Os blocos no desenho actual podem ser sobrescritos se desejado (opção "sobrescrever blocos" na barra de ferramentas das opções).
A barra de ferramentas das opções também oferece algumas ferramentas para escalar, rodar ou inverter o conteúdo da prancheta enquanto se cola.
