1. Välj önskad linjetyp i verktygsfältet för alternativ.
  2. Ange startpunkten för det första linjesegmentet. Du kan använda musen eller ange en koordinat i konsolen.
  3. Ange slutpunkten för det första linjesegmentet.
  4. Ange ändpunkterna för ytterligare linjesegment. Klicka på knappen "Close" (Stäng) i verktygsfältet för alternativ för att stänga sekvensen:

    Om du behöver ångra ett enskilt linjesegment kan du göra det genom att klicka på knappen "Undo" (Ångra)
  5. Om du vill begränsa vinkeln eller längden på linjesegmenten kan du göra det genom att klicka på knappen "Restrict Angle or Length" (begränsa vinkel eller längd) och ange en längd och/eller vinkel
