1. Após o lançamento da ferramenta, é apresentado o diálogo de texto no qual se pode escolher a fonte e introduzir o texto que se pretende criar.
  2. Escolha o nome da fonte, a altura do texto e os atributos iniciais do texto na área superior esquerda do diálogo.
    As fontes CAD que são compostas por linhas e arcos são listadas no topo da lista de fontes, as fontes TrueType são listadas abaixo.
    No campo "Altura", introduza a altura do texto na unidade do seu desenho. Para escolher o espaçamento normal das linhas, introduza um factor de espaçamento de linha ou "1". Pode aumentar ou diminuir o espaçamento entre linhas, introduzindo um valor alternativo. O valor introduzido é um factor relativo ao valor por defeito (1,0).
  3. Introduza o ângulo de rotação para o texto. Os textos horizontais normais têm um ângulo de 0 graus e os textos que são legíveis a partir dos 90 graus da direita.
  4. Escolher o alinhamento do texto.
  5. Marcar a caixa de verificação "Texto Simples" para criar um texto de linha única sem qualquer formatação avançada.
  6. Insira o texto que pretende criar na caixa de texto grande.
    Pode usar as várias ferramentas acima da caixa para formatar ainda mais o texto que introduziu ou para colar textos a partir da prancheta. Utilize o botão à direita para inserir rapidamente símbolos ou para escolher um caracter Unicode ("Outros...").
    Note que a fonte que escolheu pode não conter os caracteres que inseriu. Nesse caso, os caracteres não serão exibidos no desenho.
  7. Clique em "OK" para sair do diálogo de texto.
  8. Use o rato para especificar a localização do texto ou introduza uma coordenada na linha de comando.
  9. Muitas vezes, os utilizadores precisam de criar uma série de textos semelhantes. Por esse motivo, a ferramenta de texto não termina após a criação da primeira entidade de texto. É possível alterar o texto e o ângulo do texto na barra de ferramentas de opções e criar outros textos com configurações semelhantes.
  10. Clique com o botão direito do rato ou prima Escape para parar a ferramenta de texto.
