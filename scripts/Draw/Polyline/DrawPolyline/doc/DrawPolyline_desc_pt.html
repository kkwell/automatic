Esta ferramenta permite-lhe desenhar polilinhas. Uma polilinha é uma linha contínua formada a partir de um ou mais segmentos de linha ou arco ligados.
Note que é quase sempre mais conveniente utilizar as ferramentas de linha e arco regulares para desenhar uma nova polilinha. Os segmentos de linha e de arco livremente ligados podem então ser convertidos numa entidade polilinha utilizando a ferramenta "Criar Polilinha a partir de Segmentos".
