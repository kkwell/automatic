1. Seleccione as entidades que pretende espelhar.
  2. Lançar esta ferramenta.
  3. Especifique um primeiro ponto no eixo do espelho com o rato ou introduza uma coordenada na linha de comando.
  4. Especifique um segundo ponto no eixo do espelho.
  5. O diálogo do espelho é exibido.
    Para espelhar as entidades sem guardar as entidades originais, escolha "Eliminar Original", para copiá-las escolha "Guardar Original".
    As novas entidades são colocadas na mesma camada que os originais e têm os mesmos atributos. Para utilizar a camada actual e os atributos actuais, assinalar "Utilizar camada actual e atributos actuais".
  6. Clique em "OK" para espelhar as entidades.
