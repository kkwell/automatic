1. Seleccione as entidades que pretende mover ou copiar.
  2. Lançar esta ferramenta.
  3. Defina o ponto de referência com o rato ou introduza uma coordenada na linha de comando.
  4. Defina o ponto de destino. Para mover as entidades seleccionadas por um determinado montante, introduza uma coordenada relativa. Por exemplo, para a mover por 50 unidades de desenho à direita, introduza @50,0 na linha de comando.
  5. O diálogo de movimentação é exibido.
    Para mover as entidades, escolher "Eliminar Original", para as copiar, escolher "Manter Original". Também é possível criar qualquer número de cópias de uma só vez, escolhendo "Cópias Múltiplas" e introduzindo o número de cópias na linha de texto abaixo. Note que "9" irá criar 9 cópias e manter o original - por isso, haverá 10 instâncias das entidades seleccionadas no final.
    As novas entidades são colocadas na mesma camada que os originais e têm os mesmos atributos. Para utilizar a camada actual e os atributos actuais, assinale "Utilizar camada actual e atributos actuais".
  6. Clicar "OK" para mover ou copiar as entidades.
