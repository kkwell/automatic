1. Выберите блок, который вы хотите вставить, из списка блоков.
  2. Нажмите кнопку вставки или выберите из меню "Вставить блок".
  3. Введите угол поворота и масштабный коэффициент для опорного блока на панели инструментов опций.
  4. Укажите положение вставленного блока, щелкнув по координате или введя координату в консоли.
